# Company website for kouboulabs.com - minimal cost, high impact

**Source:** factory-new.js (interactive project intake)

## Idea

Build a high-impact company website for kouboulabs.com with minimal ongoing costs.
Domain purchased: kouboulabs.com

## Context

Koubou Labs is Xavier's venture studio / product factory.
This site should establish credibility, showcase projects, and serve as a hub for future ventures.

## Constraints

- **Budget:** Minimal cost (prefer free tier hosting: Vercel/Netlify)
- **Timeline:** ASAP (target 48h MVP)
- **Maintenance:** Low/no ongoing maintenance
- **Stack:** Static site or simple React (no heavy backend)

## Decisions (2026-03-11)

**Purpose:** Showcase portfolio projects to attract grants/partnerships
**Vibe:** Studio/agency aesthetic, warm tone
**Format:** One-pager (no navbar, single scrolling page)
**Team positioning:** Anonymous collective (no individual names exposed)
**Portfolio:** Tamago (first), more projects added going forward
**Timeline:** ASAP (48h MVP)
**Hosting:** Cloudflare Pages (free tier, custom domain, lower long-term cost than Vercel)

## Content Structure

1. **Hero** — Koubou Labs name + tagline (studio/collective positioning)
2. **Philosophy** — 2-3 sentences on approach/craft
3. **Portfolio** — Project cards (Tamago first, others TBD)
4. **About** — Anonymous team statement (we're builders, not personalities)
5. **Contact** — Email form or simple mailto link

## Tech Stack

- **Framework:** React + Vite (lighter than Next.js for static one-pager)
- **Styling:** Tailwind CSS
- **Hosting:** Cloudflare Pages (free, custom domain, generous limits)
- **Domain:** kouboulabs.com (already purchased)
- **Cost:** $0/month
